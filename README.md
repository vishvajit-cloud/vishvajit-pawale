# Student App

# Description
Student App is a web application designed to manage student information, courses, and enrollments. 
The application is connected to a MySQL/MariaDB database hosted on AWS RDS for secure data storage.

# Features
- Add, update, and delete student information
- Manage courses and enrollments
- View student details and course assignments
- Secure connection to database using SSL

# Tools
- Python (backend)
- MariaDB (database)
- HTML, CSS, JavaScript (frontend)
- Docker 
- AWS RDS (hosted database)
- Git for version control
